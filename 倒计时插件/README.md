## 相关参数

| 参数       | 作用   |类型    |  默认值 |必填 |更新 |
| --------   | -----:  |-----:  | :----:  |--- |-----|
| downList  | 需要设置倒计时的对象或者json数组 |Array/Object  |  无  |是|旧|
| start      | 开始时间的属性名 |String  |   无   |是 |当前新增|
| end        | 结束时间的属性名 | String  |   无   |是 |当前新增|


## 相关事件/函数

| 事件       | 作用     |返回值类型    | 参数值 |必填 |
| --------   | -----:  |-----:  | :----:  |--- |
| setHight   | 设置高度 |Number|  无  |否|
| success    | 处理结束后的回调 | void |   返回处理后的数据源（downList） 和高度值    |需要 时间通过这个显示倒计时 |
